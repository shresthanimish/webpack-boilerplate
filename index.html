Welcome to Webpack Boilerplate

Reference: https://www.youtube.com/watch?v=GU-2T7k9NfI

Steps

  1. npm init
    Initializes NPM project. NPM will start managing project from this point.
  2. npm install webpack --save-dev
    Installs webpack. "--save-dev" denotes that this dependency is for development only. This will not be included when packing for production.
  3. Add build scripts into package.json into script array. 
                        {
                        "name": "webpack-boilerplate",
                        "version": "1.0.0",
                        "description": "This is a boilerplate setup for webpack project",
                        "main": "index.js",
                        "scripts": {
                            "help": "echo 'Display help information (this screen)';     npm run",
                            "build": "webpack src/js/app.js dist/bundle.js",
                            "build:prod": "webpack src/js/app.js dist/bundle.min.js -p"
                        },
                        "author": "Nimish Shrestha",
                        "license": "ISC",
                        "devDependencies": {
                            "webpack": "^3.5.4"
                        }
                    }
  4. Run: npm run build npm run build:prod
    Runs npm command and builds distribution files accordingly.
